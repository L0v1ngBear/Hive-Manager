# 网页端全局青绿色主题与文字对比度设计

## 背景

管理端当前使用深蓝色 `#1f3f5f` 作为全局主色。全局兼容层同时对 `.text-primary`、`.text-on-surface` 等工具类使用了高优先级文字色，部分深色按钮、选中态和蓝色容器中的子元素因此继承或覆盖为深色文字，造成文字和图标不清晰。

本次只调整网页端视觉主题和文字对比度，不改变页面布局、业务状态颜色、权限逻辑、接口和交互流程。

## 目标

- 将网页端主色从深蓝色统一替换为常见企业软件使用的青绿色。
- 深色主题背景上的文字和图标固定使用高对比白色。
- 浅色主题背景上的文字使用深青绿或深灰，不使用低透明度主色表达正文。
- 禁用态统一使用中性灰，保留可辨识文字，不与正常主操作混淆。
- 保持错误红、警告橙、成功绿和各业务状态色的原有语义。
- 桌面端和窄屏页面都不得出现文字因主题色覆盖而不可读的问题。

## 色彩系统

| 语义 | 颜色 | 用途 |
| --- | --- | --- |
| Primary | `#0F766E` | 链接、图标、浅色选中态文字、描边 |
| Primary dark | `#115E59` | 主按钮和深色选中背景 |
| Primary hover | `#0F766E` | 主操作悬停态，保持白字 AA 对比度 |
| Primary container | `#CCFBF1` | 浅色标签、选中项、提示背景 |
| On primary container | `#134E4A` | 浅色主题容器文字 |
| On primary | `#FFFFFF` | 深色主题背景文字和图标 |
| On surface | `#0F172A` | 标题和正文主文字 |
| On surface variant | `#475569` | 次要文字和说明文字 |
| Disabled text | `#94A3B8` | 禁用文字与图标 |
| Disabled background | `#E2E8F0` | 禁用按钮和控件背景 |

## 全局实现边界

### 主题变量

在 `management-ui/src/style.css` 中统一更新 Tailwind `@theme`、`body` 下的 Hive 变量、Element Plus 主色变量以及 RGB 兼容变量。所有主色衍生值从同一套青绿色阶梯取得，避免页面内出现多套相近但不一致的主题色。

### 文字对比规则

- `.bg-primary`、`.el-button--primary`、`.function-action-primary` 和其他明确的深色主题容器，强制其直接文字、图标和加载图标使用 `#FFFFFF`。
- 深色主题容器内部的 `.text-primary`、`.text-on-surface`、`.text-on-surface-variant` 不得覆盖白色前景。
- `.text-primary` 只用于浅色或白色背景；浅色主题容器使用 `#134E4A` 或 `#0F766E`。
- 禁用态优先级高于主题态，统一使用中性灰，不使用主色透明度模拟禁用。
- Element Plus 的 primary、hover、active、disabled、loading、focus 样式与全局主题变量保持一致。

### 页面局部样式

页面内仍硬编码旧深蓝色 `#1f3f5f`、`#0b1f33`、`rgba(31, 63, 95, ...)` 的业务组件，按语义替换为全局变量或青绿色等价值。打印专用黑白样式、错误/警告/成功状态色和订单阶段色不纳入替换。

## 可访问性与验收

- 普通文字和背景的对比度目标不低于 WCAG AA 的 `4.5:1`。
- 大号或加粗文字的对比度目标不低于 `3:1`。
- 主按钮默认、悬停、按下、加载和禁用状态均可清晰识别。
- 选中标签、侧栏入口、分页当前项、复选框、单选框、开关和链接均具有清晰前景色。
- 不允许用透明度过低的主题色承载关键文字。
- 保留现有键盘焦点环和权限禁用提示，不因主题调整降低可见性。

## 测试策略

- 新增全局主题契约测试，验证青绿色变量、白色 on-primary、禁用中性色以及深色背景子元素覆盖规则。
- 扫描管理端源码中旧主题色硬编码，允许名单仅保留不属于交互主题的历史/打印用途。
- 运行管理端全部 Node 测试和 Vite 生产构建。
- 使用浏览器在订单、员工、审批、库存、安装任务、标签和首页等代表页面检查主按钮、选中态、弹窗和窄屏布局。

## 非目标

- 不改小程序主题。
- 不调整页面结构、字号体系或业务信息密度。
- 不修改业务状态色、图表数据色和打印黑白规范。
- 不新增深色模式或用户自定义主题功能。
